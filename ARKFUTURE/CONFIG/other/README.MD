# 此文件夹
* 此文件夹为所有配置的归档配置 ,即不需要的服务的配置 ,根据更新规则 我们依然会按照这些**我们不需要的服务**的**官方配置**去更新配置 ,保持配置文件是最新的 ,以便将来用得到
* HTTPS://ARKFUTURE.github.io/ARKFUTURE/CONFIG/other/
