# IRSSI 的快捷方式
```
登录服务器：/connect server port
退出irssi：/quit

窗口和频道管理快捷键

    窗口切换：
        Ctrl+n：切换到下一个窗口。
        Ctrl+p：切换到上一个窗口。
        Alt+1~0：对应1到10号窗口。
        Alt+q~p：对应11到20号窗口。

    窗口操作：
        /window new：新建窗口。
        /window close：关闭当前窗口。
        /window close [窗口编号]：关闭指定编号的窗口。
        /window list：列出所有窗口。
        /window stick [编号] off：解除窗口固定。
        /window hide：隐藏窗口。

频道管理快捷键

    加入频道：/join #<频道>
    离开当前频道：/wc 或 /part（不带频道名）
    退出频道：/part <频道> <消息>（可附加退出原因）

信息查询快捷键

    查看频道列表：/list

    列出聊天室人员：/names [#聊天室]

    查看某人资料：/whois [昵称]

    查询指定别名是否在线：/ison <别名1> <别名2> ...

    查询服务器信息：
        /info：查询当前服务器信息。
        /admin：查询当前服务器上的Admin。
        /lusers：查询当前服务器上的统计信息。
        /motd：查询当前的服务器信息（Message Of The Day）。
        /links：查询当前链接的服务器。

聊天操作快捷键

    公开消息：在频道输入框内输入完毕后直接回车即可发送。

    发送私信：
        /msg [昵称] [消息]：向某人发送私信（新开窗口）。
        /query [昵称] [消息]：向某人发送私信（新开窗口并转换到这个窗口）。
        /say [昵称] [消息]：向某人发送消息（不新开窗口）。
        /notice [昵称] [消息]：向指定人发出注意消息。

    其他聊天操作：
        /me [动作]：在当前聊天室窗口中做出动作。
        /away [原因]：设置离开状态（别人向你发出私聊时将会返回此消息）。重新输入/away（不指定参数）则解除离开状态。
        /ignore [昵称]：忽略某人的聊天内容。
        /set autolog on：自动保存聊天记录。

视图操作快捷键

    查看当前窗口中上/下页讯息：PageUP/PageDn

其他快捷键

    记录翻页：使用pgUP和pgDn键可以实现记录的翻页。
    查看历史消息：/goto 12:00 或 /goto end（跳转到指定时间或末尾），/lastlog [昵称]（查看指定昵称所说的话）。
```
