InspIRCd 服务器配置文档

安装方法

  • 官方deb包安装: install.sh
  • 全插件deb包安装: installssl.sh
  • 编译安装：make.sh
  • ARKFUTURE 定制安装包（含数据库服务和SSL）: 20250310 版本
配置测试说明：

测试时会在配置中添加 <die reason="服务器正在维护..."> 标签，该标签会阻止服务器启动，测试完成后请及时移除。

常见问题

1. systemctl 服务管理

# 获取服务模板 
cp inspircd.service  /etc/systemd/system/
systemctl daemon-reload

替代方案：直接使用 root 用户执行 inspircd 启动，通过 pkill -f inspircd 终止进程

2. 权限说明

  • NetAdmin: 服务器级管理权限
  • GlobalOp: 全局操作员权限
  • Helper: 技术支持人员权限

权限配置文件路径：/etc/inspircd/conf/opers.conf

SSL 配置

# 证书路径变量 
SSLPath = "/etc/inspircd/ssl/"
# 实际证书路径示例 
certfile = "&SSLPath;ca.crt"

推荐文档： GnuTLS 配置指南 • OpenSSL 配置指南

目录结构

/etc/inspircd 
├── inspircd.conf        # 主配置文件 
├── ssl/               # SSL证书和模块配置 
├── conf/              # 常规模块配置 
│   └── modulesconf.conf  # 插件配置文件 
└── txt/               # MOTD消息文件
